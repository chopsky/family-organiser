Star Shop premium-theme banner art.

Drop the four theme images in THIS folder with these exact filenames:

  galaxy.jpg   - space / planets / rocket (dark purple)
  dino.jpg     - dinosaurs (green)
  ocean.jpg    - underwater fish / turtle (blue)
  candy.jpg    - candy land / lollipops (pink)   <- the "Candy" theme

Notes:
- .jpg is assumed (see web/src/lib/kidsThemeArt.js). If you use .png/.webp,
  update the paths there to match.
- These render as small cover-cropped tiles on the Star Shop theme cards, so a
  modest size is plenty (~600-900px wide). The centre of each image shows, so
  keep the main character roughly centred.
- Until the files are here, the shop falls back to the old gradient swatch.
